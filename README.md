### Hex-escaper

```
Need two arguments to run the script.

Please use the hex value with double quotes! :D

USAGE:
      	./hexescaper.sh Option Hex
EXAMPLE:
        ./hexescaper.sh -e 735f796172612d72756c65736574
OPTIONS:
	-e		Escape_hex
	-r		Reverse_hex_escape
	-we		White_escape
	-wre		White_reverse_escape
```
